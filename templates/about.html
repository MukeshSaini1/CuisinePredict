Project Description

The Cuisine Classification Model is a machine learning project aimed at predicting the type of cuisine based on the ingredients used in various dishes. This project involves the following key components:

  • Data Collection: The dataset used for this project consists of various dishes and their corresponding cuisines. It is represented in a CSV file named cuisines.csv.
  • Data Preprocessing: Data is cleaned and transformed to create feature vectors for each dish. Ingredients are processed to form the basis for classification.
  • Model Training: A Logistic Regression model is trained on the preprocessed data. The model is evaluated for accuracy and performance.
  • Application: A web-based application is built using Flask and Streamlit, allowing users to predict the cuisine based on the ingredients they select.

Technologies Used

  • Python: For data processing and machine learning model implementation.
  • Scikit-learn: For building and evaluating the Logistic Regression model.
  • Pandas & Numpy: For data manipulation and numerical operations.
  • Matplotlib: For data visualization.
  • SWEETALERT: For user-friendly alert messages on the web application.
  • Flask & Streamlit: For creating the web interface for the model.
